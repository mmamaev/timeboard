Installation
************

Python version support
----------------------

:py:mod:`timeboard` is tested with Python versions 2.7, 3.6 and 3.7.


Installation
------------
::

    pip install timeboard

The import statement to run all the examples is::
    
    >>> import timeboard as tb

------------

====================================================== =================
